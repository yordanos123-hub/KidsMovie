# 🎬 KidsMovies

A simple, fun, and safe website for discovering the best animated movies for kids — built with HTML, CSS, and JavaScript.

**Live demo:** [kids-movie.vercel.app](https://kids-movie.vercel.app/)

## ✨ Features

- **Featured Movies** — highlighted picks on the homepage (e.g. Frozen, 2013 ⭐ 8.0)
- **Browse by Category** — Disney, Pixar, Anime Kids, Educational, or view all
- **Search** — quickly find a movie by name
- **Simple, clean UI** — no clutter, easy for parents and kids to navigate
- **Responsive layout** — works on desktop and mobile

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript
- Hosted on [Vercel](https://vercel.com/)

## 🚀 Getting Started

Clone the repo and open it locally:

```bash
git clone https://github.com/yordanos123-hub/KidsMovie.git
cd KidsMovie
```

Since this is a static site, you can simply open `index.html` in your browser, or serve it locally:

```bash
npx serve .
```
